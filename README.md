🎥 Live Object Detection & Tracking

Description:
This is a simple Streamlit app that can detect objects using a camera. It uses YOLOv8 model.

Features:
* Detect objects
* Shows boxes and labels
* Uses webcam

Technologies Used:
* Python
* Streamlit
* streamlit-webrtc
* YOLOv8 (Ultralytics)
* OpenCV
* AV 

Run the app:
python -m streamlit run app.py
